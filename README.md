## Description

Its a simple game where you move the blob onto the levels using the arrow keys or the "W", "A" , "D" keys. The goal is to avoid the spikes that randomly pop up on each level. When you get hit by the spikes the game restarts.

---

## Learning Goals

Learning Goals:

- load JSON in preload()
- create WorldLevel from JSON
- create BlobPlayer
- update + draw each frame
- handle input events (jump)
